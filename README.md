## Local Development Setup

Prerequisites: Node.js

1. `gem install sass`
1. `npm install`
1. `grunt`

## Local Development Workflow

1. `npx grunt watch`
1. Open `http://zachleat.localhost/`

To install new local npm packages, use `npm install PACKAGE_NAME --save-dev`

## Deploy

1. `npx grunt deploy`
